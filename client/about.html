Just My Type is an interactive typing game to measure and improve your typing speed and accuracy

Instructions:

First pick how many sentences you would like to practice with and then the difficulty level. The difficulty setting is based on the length of the words to dynamically be displayed- it is pulled from a 20,000 word dataset. Afterwards, press enter! The timer will start and the first sentence will appear. The letter you are currently targeting will be highlighted until successfully matched. Upon completion your WPM will be calculated. Practice makes perfect!

How I Developed This

Font-end: HTML, CSS (Bootstrap) and Javascript (jQuery)

I would strongly recommend anyone new to JavaScript try and recreate this application. Oftentimes those new to coding learn all the basics of the language and then hit a brick wall- they understand the fundamentals of the language but don't see a practical way to get up and running with their first application. This app will strengthen your DOM Manipulation skills, along with getting one in the mindset of thinking like a programmer. My github profile has a detailed instruction README that will provide an outline to completing the project on your own!
